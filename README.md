# Block Sandbox Game (Ursina Engine)

A simple Minecraft-inspired sandbox built with [Ursina](https://www.ursinaengine.org/) in Python.  
This project was created for fun to explore block placement, destruction, and first-person controls.

## Features
- First-person movement (WASD keys)
- Jump with Shift
- Place blocks with left mouse click
- Destroy blocks with right mouse click
- Grass-textured cubes arranged in a grid
- Infinite gravity (fall endlessly) â€” intentional design choice
- Optional walls can be added if needed
